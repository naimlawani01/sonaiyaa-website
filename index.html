Comment ça marche

01

Créez votre commande

Renseignez le nom du client, le contenu du colis et l'adresse. Le prix se calcule automatiquement.

En quelques secondes, un livreur vérifié proche de vous accepte la course. Votre client reçoit un SMS de suivi.

Le livreur livre, le client confirme avec son code PIN. Vous recevez la confirmation instantanément.
