Comment ça marche

Trois étapes. Pas une de plus.

  1. 01

    Créez la commande.

    Nom du client, contenu du colis, adresse de livraison. Le prix est calculé automatiquement selon la distance. Vous validez en 30 secondes.

  2. 02

    Un livreur accepte.

    Notre système matche votre commande avec le livreur le plus proche, en quelques secondes. Le client reçoit un SMS avec un lien de suivi GPS — il ne télécharge rien.

  3. 03

    Livré, confirmé, payé.

    Le livreur arrive. Le client confirme avec un code PIN à usage unique. Vous recevez la notification, le paiement est encaissé. C'est tout.
